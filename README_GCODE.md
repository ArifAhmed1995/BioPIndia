## README for GCode Branch

`Slic3r` will be used for translating the 3D Model to GCode.

BioP software will be responsible for parsing the GCode and translating it
to commands which the firmware can interpret.

A GCode file generated by Slic3r will have a specific set of commands for a given interpreter set-up.
If any other generator other than Slic3r is used, most likely the interpreter will need to be changed as well.
